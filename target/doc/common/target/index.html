Module target

Source
Expand description

Target configuration utilities. Handles configuration related to LLVM target machines, including setting up and querying target-specific information. Module for handling LLVM target configurations.

This module provides functionality to configure different processor targets for LLVM, so that we can generate code for a wide variety of architectures from the same Intermediate Representation (IR). Each target configurator is responsible for initializing the necessary components of LLVM for a specific architecture.
